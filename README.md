# PowerApi

PowerApi是一个非常适合IT团队在线管理API项目的工具，它可以进行数据接口的内容管理、单元测试、文档生成、接口使用状态标记以及自动化测试、自动化监控等功能。

## 技术选型
* Spring 4.2.6
* Mybatis Plus 2.0
* Shiro 1.2.2
* Hibernate Validator 5.1.3
* Druid 1.0.19
* Redis 2.3.1
* Bootstrap 3
* JDK 1.7
* Tomcat8
* Maven
